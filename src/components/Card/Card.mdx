Um Card é um componente para exibir informações.

## Como usar

Este componente Card foi projetado para ser exibir informações da população de um determinado bairro.

Aqui está um exemplo de um Card.

A prop `population` (obrigatório) é uma lista de objectos que possuem o ano e o tamanho da população naquele determinado ano .

```html
<Card
  :population="[
    { ano: 2000, populacao: 1000 },
    { ano: 2010, populacao: 2000 },
    { ano: 2020, populacao: 3000 },
    { ano: 2030, populacao: 4000 },
  ]"
/>
```
